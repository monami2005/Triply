# 🚀 TripSplit Pro v2.2
### Expense Splitting for Travel Groups - Premium SaaS Version

**TripSplit Pro** is a high-end, full-stack web application designed for travel groups to manage and split expenses with elegance and precision. Featuring a state-of-the-art **Glassmorphism UI**, smooth **GSAP animations**, and a robust **WhatsApp sync engine**.

---

## ✨ Premium Features
- **💎 Glassmorphic Dashboard**: A stunning visual overview of trip stats, recent activities, and group balances.
- **📅 Date-wise Explorer**: Track your spending pattern with a powerful daily expense filter.
- **📲 WhatsApp Integration**: One-click bulk messaging to send payment summaries to all members using customizable templates.
- **🌗 Smart Theming**: Support for Light, Dark, and System-synced themes with dynamic UI transitions.
- **📸 Member Profiles**: Manage your squad with photo avatars and detailed individual spending reports.
- **⚡ Pro Split Engine**: Intelligent expense distribution with multi-select participants and per-head calculations.

---

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla JavaScript (ES6+), GSAP (Animations), FontAwesome 6, Google Fonts (Outfit).
- **Styling**: Modern CSS (Variables, Glassmorphism, Flex/Grid).
- **Backend**: Python 3, Flask (Vercel Serverless Compatible).
- **Database**: SQLAlchemy with SQLite (Local) / PostgreSQL (Production).

---

## 🚀 Deployment to Vercel
This project is built to be **Vercel-Ready** out of the box.

1. **Push to GitHub**: Upload your project to a GitHub repository.
2. **Import to Vercel**: Connect your repo to Vercel.
3. **Configure**:
   - Vercel will automatically detect the `vercel.json` and `api/` folder.
   - For production database, set `DATABASE_URL` as an environment variable (e.g., Supabase or Neon PostgreSQL).
4. **Deploy**: Hit deploy and enjoy your premium SaaS!

---

## 💻 Local Setup
1. Ensure you have **Python 3.x** installed.
2. Clone the repository.
3. Run `run_locally.bat` to automatically install dependencies and start the server.
4. Open `http://localhost:5000` in your browser.
